# LLM Prompts Documentation

## Prompt Engineering for AI Summary Generation

This document contains the prompts used for the AI summary generation feature.

## AI Summary Prompt

### System Message
You are a helpful AI audit assistant. Your role is to analyze the AI tool spending data and provide a concise, personalized summary for the user based on their audit results.

## Prompts

### Summary Generation Prompt
Create a personalized summary of the AI tool audit results that explains:
1. Whether the user is currently overpaying for their AI tools
2. How much they could save by switching tools or plans
3. Recommended actions to optimize their AI spending
4. The potential savings from reviewing their current AI tool subscriptions

The summary should be professional and helpful, explaining the potential savings and recommendations in a clear format that helps the user understand their AI spending.

## Prompt Guidelines

The prompt should:
1. Be clear and concise
2. Focus on the key findings from the audit
3. Provide actionable recommendations
4. Not be too technical in language
5. Be formatted for easy reading

## Example Prompts

### Summary Generation Prompt
"Analyze the AI tool usage and provide a summary of potential savings for the user. The summary should explain:
1. Current tools and spending
2. Potential savings from optimization
3. Recommendations for cost reduction
4. Comparison of current plan to alternatives
5. Total potential savings calculation"

This prompt should generate a clear, concise summary that helps the user understand their current AI tool usage and potential savings.
